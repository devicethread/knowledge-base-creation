This article covers in details all the check list of setup steps that you should complete in Devicethread before connecting the Cloudbed Integration with the Devicethread app.

Make sure you complete the following 4 steps:

  1. Download the SmartThings App and Create a Samsung Account. One account should be used for each property.

  2. Set up One or More SmartThings Hubs on the Property.
    1. Please note that each SmartThings hub has a range between 50-100 feet based on having a line of sight.

  3. Set Up One or Multiple Smart Locks on the Property - For guest rooms and common doors such as the front door and back door.

  4. Do the Initial Onboarding Steps in the Devicethread App - Before connecting Cloudbeds & Devicethread - please do these steps in the order called out below.

Get the Devicethread app and create an account.


  • Create a Devicethread account.


  • Create a "Property" enter a name, physical address, time zone, and add a photo.



Connect your property's SmartThings account to Devicethread. Go to the "More" page under "Integrations & Cloud Accounts" section, click SmartThings.

  • Click the + button to add your SmartThings account


  • Click "Generate new Token" and follow the instructions in the app.


  • Sign in to your Samsung account and click "Next"

  • Click "Generate new token"

  • Select all of the Authorization Options (Full scope) by checking all the boxes. We recommend you do so to enable all the Devicethread capabilities in the app.

  • Click "Generate Token"

  • Your new SmartThings token has been generated.

    Copy this new token in a safe place as your will not be able to see it again. If you accidenlty loose it, you will have to generate a new one in the future as it cannot be retrieved

  • Go back to the "More" page under the "Integrations Cloud Accounts" section, click SmartThings.

  • Click the + button to add your SmartThings account.

  • Click "Paste an existing Token" since you just created one and copy it to the clipboard.

  • Enter the email address you used to create the SmartThings account and enter or paste the SmartThings API Token you have just created.

  • Click "Connect Account Now".

  • Well done! Devicethread confirms your account has been connected successfully.

Now it is time to create "Zones. Create all the necessary "Zones" to map out your property in Devicethread.

A "Zone" is a defined physical area on the property like "Room 101" where one or more "devices" like door locks are located. Each door lock must be assigned to a "Zone" and multiple door locks can be assigned to a "Zone". For example, a room might have 2 smart door locks located in it.

  1. Go to the "More" page

  2. Click "Zones" - Yes, there are none for now...

  3. Click "Create First Zone" str for each smart lock location where each smart lock is installed (name of the physical location)

  • Give each "Zone" a name and select the "Type" that best describes the "Zone" in question:

    • Wing, Floor, Room, Guest Room, Entire Home, Other. More options are being added regularly to better fit your property.

    • Click "Create Zone" then Success!

  • Click the new "Zone" name you just created on the "Manage Zones" page.

  • On the "Devices" tab, click "Add Devices". You will now add all the door locks or devices that are in the specific "Zone".

  • Select all the appropriate locks by checking the boxes under the "Not Added", which means the locks have not been assigned yet.

  • Click "Add Selected Devices"

  • Devicethread confirms the "Zone" has been accessed successfully.

  • All the "Zones" you have created will now show up on the "Manage Zones" page, and you can access them at any time to manage them.

Add Property Users: all the initial employees and workers that work at the property are assigned as "Users" at this property.

  • Go to the "More" page and click "Users" under "User Management".

  • On the "Users" page, click on the + button to add new users to the property highlighted on the top of the page.

  • Fill out the required user information to assign them to a property.

    • Select the "Organization" they belong to.

    • Select their name from the pull-down menu. Please note that a user can only be added to a property if they first belong to an organization related to the property, such as the owner group or property management company. If not, their name will not appear on the pull-down menu.

  • Assign each "User" a "Role" which defines their access to specific zones and permissions for the property (Property Manager, Property viewer, Property Engineer, Front Desk Admin, Front Desk, and Staff).

  • Assign each "User" a "Group" such as: Engineering, Front Desk, Housekeeping, and Management to help organize users into groups with similar responsibilities at the property.

Define some key settings for the property.

  • Go to the "More" page and click "Preferences"

  • Click "Battery Alert"

  • Set the Low Battery Level notification: you decide when you want Devicethread to notify you that the batteries of a particular lock are running low. We recommend 25%.

  • Click "Lock Code Format" also under "Preferences". Decide the length of the lock codes to be generated by Devicethread: 4-digit or 6-digit codes.
